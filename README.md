# Homestay (JFSD Project)

A small application for browsing and managing homestay listings built for the JFSD coursework.

## Tech Stack
- Frontend: React <JS/TS>, React Router, Axios
- Backend: <Spring Boot / Express.js> with REST APIs
- Database: <MySQL / MongoDB>
- Dev: Node.js, npm/yarn, <Maven/Gradle>, Git

## Features
- Listings: create, read, update, delete
- Search & Filters: by location, price, capacity
- Booking: request/reserve flow (basic)
- Auth: <session/JWT> (minimal)
- Validation & Error Handling: server & client side
- Responsive UI
